GitHub Contribution Painter

Рисует паттерн на GitHub графе через API — без терминала. Нужен токен с правом repo и пустой репо с README.

Настройки
Создать токен — нужна галочка repo.
Должен существовать и содержать хотя бы один коммит (README).
Проверить email — должен совпадать, иначе граф не обновится.
