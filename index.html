GitHub Contribution Painter

Рисует паттерн на GitHub графе через API — без терминала и скриптов.
Нужен Personal Access Token с правом repo и пустой репозиторий.

Настройки
Создать токен — нужна галочка repo. Токен не сохраняется и не передаётся никуда, кроме api.github.com.
Репо должно существовать на GitHub и быть пустым.
Больше коммитов = ярче зелёный цвет на графе.
Редактор
Интенсивность:
клавиши 0–4  |  зажми и веди мышью для рисования
Закрашено дней: 0
Коммитов всего: 0
